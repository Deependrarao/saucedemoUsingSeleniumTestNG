# SauceDemo Automation Framework (Selenium + TestNG)

This project is a **Test Automation Framework** built using:
- Java
- Selenium WebDriver
- TestNG
- Maven

It automates the core functionalities of the SauceDemo website like login validation and basic user flow.
 Application Under Test: https://www.saucedemo.com/
---
## Tech Stack

| Tool | Purpose |
|------|--------|
| Java | Programming Language |
| Selenium WebDriver | UI Automation |
| TestNG | Test Execution Framework |
| Maven | Build Management |
| POM | Design Pattern Used |

---

## 🧱 Framework Design

This framework follows:

✅ Page Object Model (POM)  
✅ Reusable Utilities  
✅ TestNG Execution  
✅ Maven Build Support  

POM helps keep test logic separate from UI locators, improving maintainability and scalability.

---

##  Prerequisites

Make sure you have installed:

- Java 17+
- Maven
- Chrome Browser
- IDE (IntelliJ / Eclipse)

Check installation:
bash
java -version
mvn -version

## The framework uses:

Page Object Model (POM) for element abstraction
TestNG for grouping, running, and reporting tests
Selenium WebDriver for automation
Maven for dependency & build management
